1. Создаем проект.
2. Инициализируем Git в проекте(в терминале пишем): git init
3. Устанавливаем локальные настройки(пишем в терминале):
    a. git config user.name "Ваше логин"
    b. git cinfig user.email "Ваш email"
4. Добавляем файл в проект .gitignore, в этом файле прописываем:
    a. *.sass
    b. bin/
    c. admin/
    d. config/
5. Добавляем фалы в локальное хранилище(пишем в терминале):
    а. git add . - добавит все файлы в режим ожидания
    b. git commit -m "Комментарий к проекту"
